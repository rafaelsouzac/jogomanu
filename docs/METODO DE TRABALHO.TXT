3) Todo projeto é feito com base em sistemas:
    Login é um sisma então:
        controller/LogingController.php --> contém todos os dados necessários para
                                                  gerar a página home.php conforme deve aparecer o sistema login.

        model/LoginModel.php        --> faz acesso ao banco de dados e envia os dados
                                                  ao script de controller.

        view/assets/partes/login_base.php 
            --> Mantem todos os dados necessários para formar a parte da base da pagina do sistema login
        
        view/assets/partes/login_centro.php
            --> Mantem todos os dados necessários para formar a parte da base da pagina do sistema login

        view/assets/partes/login_topo.php
            --> Mantem todos os dados necessários para formar a parte da base da pagina do sistema login

            app.css --> contem todas regras utilizadas em todas as páginas da aplicação

        js/ --> mantem os seguintes arquivos:
            app.js --> contem todos os scripts utilizados em todas as páginas da aplicação
            <nome_pagina>.js --> contem todas regras utilizadas somente na página
        
        home.php --> E a página apresentada ao usuário. Contém todo html utilizado em todas as página da aplicação e monta a parte dinâmica com require_once() aplicado de /view --> base.php,  centro.php, topo.php
